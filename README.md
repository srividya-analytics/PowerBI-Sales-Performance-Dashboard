# 📊 Power BI Sales Performance Dashboard

## Project Overview

This project presents an interactive Sales Performance Dashboard developed using Microsoft Power BI. The dashboard provides insights into sales performance, profitability, customer purchasing behavior, product performance, and store-level sales using interactive visualizations and business-focused KPIs.

The objective of this project was to transform raw sales data into meaningful business insights through data cleaning, modeling, DAX calculations, and dashboard development.

---

## Business Problem

Businesses generate large volumes of sales data every day. Without proper analysis, it becomes difficult to identify top-performing products, profitable categories, customer trends, and sales patterns.

This dashboard helps decision-makers quickly monitor key business metrics and make data-driven decisions.

## Power Query Transformations

The dataset was prepared using Power Query by performing:

* Data cleaning
* Removing duplicates
* Handling missing values
* Data type corrections
* Creating calculated columns
* Preparing data for analysis

---

## Data Modeling

A Star Schema data model was implemented to improve performance and simplify analysis.

Tables included:

* Orders (Fact Table)
* Customers
* Products
* Stores
* Date Table

Relationships were created between dimension tables and the fact table to support interactive filtering.

---

## DAX Measures

Several business measures were created using DAX, including:

* Overall Sales
* Total Profit
* Total Orders
* Profit Margin %
* Average Order Value
* Sales by Category
* Net Sales
* High Value Sales
* Profit Status
* Discount Status

The project also includes logical DAX functions such as IF(), SWITCH(), VAR(), CALCULATE(), SUM(), COUNTROWS(), DIVIDE(), and AVERAGE().
